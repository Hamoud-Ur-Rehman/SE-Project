 PlayDB — Guided by Taste
A web-based video game tracking and marketplace platform built for Pakistani gamers. Track your library, manage lists, connect with friends, and buy/sell gaming gear — all in one place.

 How to Run
Ensure all files and folders are present. Ensure Maven is installed. Navigate to the directory which contains pom.xml. Enter the Command Prompt and type
mvn spring-boot:run
If the command is executed without errors and shows “Application started in x seconds”, open a search engine, and type as the URL
localhost:8080 
This allocates port 8080 to our process, and the user can use PLAYDB
The benefit is that no IDE is required, just a simple search engine

 Key Features
•	User Auth — Secure signup, login, and session management with encrypted passwords
•	Game Library — Browse a game database and add titles to personal lists (Currently Playing, Completed, and more)
•	Marketplace — Post and browse ads for physical and digital gaming products (Pakistan only)
•	Social & Chat — Send friend requests, discover other players, and message directly in-app
•	RecBot (coming soon) — AI-powered game recommendations based on your play history
